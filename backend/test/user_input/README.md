# 사용자 입력 이미지 (음식 사진)

파이프라인의 핵심 입력입니다. **레퍼런스 이미지에 이 음식이 적용된 결과물을 생성하는 것이 목표입니다.**

이 폴더에 실험용 음식 사진을 넣어주세요.

| 파일명 | 설명 |
|--------|------|
| `product_01.png` | 사용자 입력 이미지 1 |
| `product_02.png` | 사용자 입력 이미지 2 |
| `product_03.png` | 사용자 입력 이미지 3 |
