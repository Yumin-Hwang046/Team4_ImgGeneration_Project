## ✅ DALL·E 3 대체안 (Hugging Face, 품질 A, GCP L4 기준)

DALL·E 3 접근 권한이 없을 때 Case 1(텍스트 → 이미지)을 대체하는 모델 리스트입니다.  
**품질 우선(A)** 기준이며 **GCP L4 1장**을 전제로 합니다.

### 🔥 추천 모델 (품질 A)
1. `stabilityai/stable-diffusion-xl-base-1.0` + `stabilityai/stable-diffusion-xl-refiner-1.0`
   - SDXL 표준 조합. 품질 안정적이고 범용 텍스트 → 이미지에 가장 무난
2. `SG161222/RealVisXL_V5.0`
   - 실사형 광고 이미지에 강한 포토리얼리즘 모델
3. `segmind/SSD-1B`
   - SDXL 계열 경량 모델. 속도/자원 효율은 좋지만 품질은 1번보다 낮음

### ✅ 품질 A 기준 기본 세팅 (L4 최적)
- Base + Refiner 2단계 생성
- Steps: 30 (Base), 15 (Refiner)
- Guidance: 6.5~7.5
- 해상도: 1024 계열 (SDXL 권장)
- dtype: fp16 (GPU 필수)
